# MovieFlix

A movie search app built with Next.js and Tailwind CSS, powered by the TMDB API.

## Live Demo
Coming soon

## Pages
- Home — trending movies
- Search — search movies by name
- Movie Detail — full movie info
- Favorites — saved movies (localStorage)

## Tech Stack
- Next.js 15
- Tailwind CSS
- TypeScript
- TMDB API
- React Icons

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Add `.env.local` with `NEXT_PUBLIC_TMDB_API_KEY=your_key`
4. Run `npm run dev`
